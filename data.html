Our Assembled Dataset


Data Entry and Cleaning Process

Using the manifests we obtained from the EPA, we assembled a spreadsheet of data that included importer names and locations, exporter names and locations, various EPA and United Nations codes, waste type, waste amounts, and units the waste was reported in, along with port of entry city and any discrepancies that were either noted on the manifest or we noticed were there upon entering the data into our spreadsheet. The dataset we have assembled to-date can be downloaded here.

Before we could use our dataset to create maps and other visuals, we had to clean the dataset. Below are the steps we took to process the data.

  1. Clear 2006 and 2014 data because we only have a sample of that data so far
  2. Clear anything without a quantity
  3. Convert quantities with (EST) or (APPROX) to the value that comes before (EST) or (APPROX)
  4. Clear anything without a unit
  5. Clear anything without an exporter address
  6. Clear anything without an importer address, which was rare if any
  7. Standardize importer addresses to match a lookup table we created (e.g., 625 WANDA STREET should be 625 WANDA ST)
  8. Lookup importer latitude and longitude values
  9. Lookup exporter latitude and longitude values
  10. QA/QC any latitude and longitude errors
  11. Standardize EPA keys
  12. Create exporter keys (E + latitude first two digits + A + longitude last four decimal digits)
  13. Convert units to kilograms and liters
  14. Clear anything without a waste description AND UN code
  15. Lookup UN codes
  16. If there was a waste description but no UN Code, UN Code Description = Not provided
  17. If there was a UN Code but no waste description, UN Code Description = Unknown
  18. Lookup expected management codes
  19. If no management code, then we indicated that it was "Not provided"
  20. If multiple management codes (e.g., H040;H141), then "Multiple management methods." was indicated
  21. If a code was not known as an EPA or UN code (e.g., "A2342"), then classified as "Unknown"
  22. Lookup company
  23. Map the facilities
  24. QA/QC latitude and longitude errors (e.g., make sure a Texas facility is not given the coordinates of an Ohio facility)
  25. The dataset is now ready to be used!
